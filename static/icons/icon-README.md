# PWA Icons for Universal ID Card Generator

## Icon Files

### Standard Icons (purpose: "any")
- **icon-192.png** (192x192px) - Used for home screen on mobile devices
- **icon-512.png** (512x512px) - Used for splash screens and larger displays

### Maskable Icons (purpose: "maskable")
- **icon-maskable-192.png** (192x192px) - Adaptive icon for Android devices
- **icon-maskable-512.png** (512x512px) - Large adaptive icon

## Design Details

**Icon Style:** Modern, professional gradient design
- **Primary Color:** Purple (#667eea) to Deep Blue (#764ba2)
- **Design Elements:** ID card with security checkmark badge
- **Style:** Minimalist, bold geometric lines
- **Theme:** Professional, trustworthy, secure

## Usage

These icons are configured in `manifest.json` and automatically served as:
- Home screen icons on mobile devices
- App launcher icons
- Splash screen backgrounds
- Browser tab icons (via favicon)

## Technical Notes

- All icons use PNG format for transparency support
- Icons are optimized for both regular and high-DPI displays
- Maskable icons designed for safe zone compatibility with all app launchers
- Icons are resolution-independent and scale cleanly at any size
